dnd-classes
==============

An interactive HTML quick reference hub for D&amp;D 5e (2024 rules / 5.5e) classes. 

This project is designed to help players quickly access and understand their class's core features, combat roles, and weapon choices during gameplay without having to flip through rulebooks. It features a clean, RPG-themed design and allows users to seamlessly switch the interface between multiple languages. Includes a main hub and individual class pages (currently Monk and Fighter).


Project status
==============

This repository has been updated for the **2024 edition** of the D&amp;D 5e rules.
It also includes internationalization support for Brazilian Portuguese (pt-BR), English (en), and Spanish (es).
